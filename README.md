## 第一卷

在上帝搜索的现状中，奥古斯丁着手探寻他的记忆深处，追溯他生命已走过的神秘的恩典朝圣之旅---并为祂恒常而万能的恩典赞美上帝。在一种经久祷告的情绪中，他回忆他所能回忆的幼年、他学习说话以及他在学校的童年经历。他以一首对上帝感激的赞美颂歌来结尾。
